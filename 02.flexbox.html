Flexbox Properties

1.Justify-content...

Justify content property aligns the items in the horizontal direction in main axis.There is other things like flex start to align items to the left side of the container,flex end to transfer the items to the end of the container, flex center to transfer the items to the center of the container,space betwwen is used to display the items with equal spacing between them,space around to display the items with equal spacing around them.

2.Align-items...

Align-items property aligns the items in the vertical direction in cross axis.There is other things like flex start to align items to the left side of the container,flex end to transfer the items to the end of the container, flex center to transfer the items to the center of the container,space betwwen is used to display the items with equal spacing between them,space around to display the items with equal spacing around them.

3.Gap...

The gap property defines the size of the gap between the rows and betwwen the columns in flexbox,grid or multi-column layout.It is a shortand for the following properties:1.row-gap 2.column-gap

4.Flex-direction...

The flex-direction property specifies the direction of the flexible items.In flex-direction has two direction row and column.In flex direction have a reverse option to reverse the items in the container.By default the flex direction is row to move the items in horzintal or vertical direction.In the flex direction to choice the row direction the main axis is horizontal(Justify-content) and cross axis(Align-items) is vertical but the flex direction to choice thre column direction the main axis is vertical(Justify-content) and cross axis is horizontal(Align-items).

5.Flex-wrap...

The Flex-wrap property specifies whether the flexible items should wrap or not.flex wrap are some property.

  • flex-wrap:nowrap;
  • flex-wrap:wrap;
  • flex-wrap:wrap-reverse;
